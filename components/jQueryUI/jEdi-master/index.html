Editor buttons:

Set data-cmd and data-value attributes to assign editor commands:

  • Bold
  • Italic
  • Underline
  • Font-Size = 5
  • Alert Content

you can find a list of supported commands at mozilla developer network:
https://developer.mozilla.org/en-US/docs/Rich-Text_Editing_in_Mozilla#Executing_Commands
